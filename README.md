# Beginner API Testing with Postman

## Tools Used
- Git
- GitHub
- Postman

## APIs Tested
- https://jsonplaceholder.typicode.com/

## Requests Included
- GET /Todos
- GET /Todos/1
- POST /Todos
- DELETE /Todos/1

## Steps to Run
1. Clone the repository
   git clone https://github.com/codewithprajapti/api-testing-postman.git

2. Open Postman

3. Import Collection
   postman/Beginner-API-Testing.postman_collection.json

4. Import Environment
   postman/environment.postman_environment.json

5. Select environment and run requests
